🔥 Configuration Firebase

Firebase est la base de données réelle qui stocke vos comptes et parties. La configuration est faite une seule fois.

  1. Allez sur console.firebase.google.com
  2. Créez un projet → Activez Firestore et Authentication
  3. Dans Auth → Sign-in method → activez Email/Password
  4. Dans Project Settings → copiez la config Web
  5. Collez la config JSON ci-dessous
